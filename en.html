Privacy Policy

Last Updated: February 19, 2026

1. Data Collection

This application does not collect any personal information, usage data, or identifiers. No account creation or login is required.

2. Data Processing

All data processing is performed solely on your device. This application does not transmit data to external servers for processing.

3. Local Access

This application requests access to your photo/video library solely for the purpose of playing the videos you select.

  • Access is only granted with your explicit permission.
  • Photos and videos are not used for analysis, advertising, or any other purpose.
  • Photos and videos are never transmitted to external servers.

4. Third-Party Sharing

This application does not transmit or share any data with third parties. It does not contain analytics tools or advertising SDKs.

5. GDPR and CCPA Compliance

GDPR: As this application does not collect or process personal data (Personal Data), obligations as a Data Controller or Data Processor do not apply. However, it is designed to respect user privacy fully.

CCPA: As this application does not collect or sell consumer personal information (Personal Information), opt-out procedures under CCPA are not required.

Users can delete all associated data at any time by uninstalling the application from their device settings.

Contact

For questions regarding this privacy policy, please contact us via the GitHub repository Issues or contact the developer directly.
