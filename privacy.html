Local Video Shuffle Privacy Policy

Last Updated: July 4, 2026

This policy applies to Local Video Shuffle for macOS (Mac App Store and direct-distribution builds).

1. Data Collection

This application does not collect any personal information, usage data, or identifiers. It does not collect, sell, or share personal data. No account creation or login is required, and it contains no advertising, tracking, or analytics.

2. Data Processing

All data processing is performed solely on your device. This application does not transmit data to external servers for processing.

3. Local Video Folder Access

This application requests access to your local video folders solely for the purpose of playing the videos you select.

  • Access happens only when you explicitly choose a folder.
  • Videos are not used for analysis, advertising, or any other purpose.
  • The app does not upload user videos to external servers.

4. Private Vault

Private Vault stores imported videos as encrypted local files in the app's data area. Private Vault metadata is also encrypted locally.

On macOS, the vault key is protected through the system Keychain when the native helper is available. Unlocking the vault uses the standard macOS authentication flow, such as Touch ID or device password, depending on the user's Mac and system settings.

Deleting the Private Vault from inside the app removes the encrypted vault files and vault key material. Deleting only the app executable or app bundle may not run app code, so the app cannot guarantee cleanup in every unmanaged deletion path.

5. Network Access

Mac App Store builds receive updates through the App Store.

Non-App-Store builds may contact GitHub Releases to check whether an update is available. This check does not upload user videos.

6. Third-Party Sharing

This application does not transmit or share any data with third parties. It does not contain analytics tools or advertising SDKs.

7. GDPR and CCPA Compliance

GDPR: As this application does not collect or process personal data (Personal Data), obligations as a Data Controller or Data Processor do not apply. However, it is designed to respect user privacy fully.

CCPA: As this application does not collect or sell consumer personal information (Personal Information), opt-out procedures under CCPA are not required.

You can change which local folders to open at any time, delete Private Vault data from inside the app, and remove remaining data by deleting the app and its data area (app container).

Contact

For questions regarding this privacy policy, use the GitHub Issues link on the Support page or contact the developer directly.

Terms of use and disclaimers are in the Terms of Use.
