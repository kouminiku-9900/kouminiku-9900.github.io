Local Video Shuffle Support

Local Video Shuffle plays user-selected local video folders and includes an encrypted Private Vault for selected videos.

Common Checks

  • If a folder no longer opens after restart, choose the folder again so macOS can refresh permission.
  • If a video will not play, convert it to MP4 using H.264 video and AAC audio.
  • If Private Vault unlock fails, confirm that protected storage is available for the current macOS user. If a Touch ID prompt appears, complete it and try again.

Contact

For issues, feedback, and feature requests, email kouminiku@gmail.com. This is the functional support contact available to all users.

When reporting a problem, include the app version, macOS version, what you expected, what happened, and the steps needed to reproduce it. Screenshots and crash logs are helpful, but remove personal information and unnecessary local file names before attaching them.

Security and privacy reports

For a possible Private Vault vulnerability, unintended decryption, or disclosure issue, do not post details publicly. Use the subject “Security Report” and report it privately by email. Do not send actual sensitive videos, vault keys, passwords, or personal data.
